# Databricks Medallion Architecture

## Overview
This project demonstrates a scalable ETL pipeline using PySpark and Delta Lake.

## Architecture
Bronze → Raw Data  
Silver → Cleaned Data  
Gold → Aggregated Data  

## Tech Stack
- PySpark
- Delta Lake
- Databricks

## Features
- Incremental processing
- Data cleaning
- Aggregation

## How to Run
1. Install PySpark
2. Run bronze.py → silver.py → gold.py
